[344] Reverse String

难度：Easy
语言：C++

题目描述。
左右指针往中间靠拢即可，无难度。

  void reverseString(vector<char>& s) {
    char tmp;

    for (int left = 0, right = s.size() - 1; left < right; left++, right--) {
      tmp = s[left];
      s[left] = s[right];
      s[right] = tmp;
    }
  }
